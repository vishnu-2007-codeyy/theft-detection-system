# 🚨 Theft Detection System with Telegram Alerts

A real-time theft detection system using computer vision that sends instant alerts with photos to your Telegram when intruders are detected.

## ✨ Features

- 📸 **Real-time face detection** - Identifies human faces in video feed
- 🟢 **Motion detection** - Green boxes highlight movement
- 🔴 **Intruder alerts** - Red boxes mark detected faces
- 📱 **Telegram integration** - Instant photo alerts on your phone
- ⏱️ **Adjustable cooldown** - Prevents notification spam
- 💾 **Auto photo saving** - All intruder images saved locally
- 🧪 **Test mode** - Press 't' to send test alert

## 📋 Requirements

- Python 3.7+
- Webcam
- Telegram account
